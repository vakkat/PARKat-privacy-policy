# PARKat Website

Official website for the PARKat mobile application.

This repository hosts the public website (via GitHub Pages) for:

- Download instructions
- User guide
- Privacy Policy
- Research information

## About PARKat

PARKat is a research-based mobile application developed by Katerina Vakrinou, Ph.D. Candidate @ NTUA.

The app collects mobility and parking-related data to analyze parking search behavior in urban environments.

## Website Structure

The website includes:

- `index.html` – Home page
- `download.html` – Installation instructions
- `guide.html` – User guide
- `privacy.html` – Privacy Policy
- `styles.css` – Styling 
- `app.js` – Language toggle (Greek / English)

## Deployment

This website is deployed using **GitHub Pages**.

## Research Context

The PARKat application is used for academic research purposes in the field of:

- Urban mobility
- Parking behavior
- Intelligent transportation systems

For more information, visit the live website.
